# Economy (WIP)

An advanced Economy plugin for the Bukkit API.

- [X] Vault.
- [X] MongoDB connection.
- [ ] Redis implementation.
- [ ] BungeeCord implementation.
- [X] Admin give, balance and take command.
- [X] /balance command.
- [X] /pay command.
- [ ] /payhistory command.
- [ ] /payrequest command.
- [X] Auto-translation to player client language.
